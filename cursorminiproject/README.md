# CrowdWatch — Crowd Density Estimation for Public Safety

A modern website showcasing a crowd density estimation system designed for public safety management.

## Run Locally

Open `index.html` directly in your browser, or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Structure

- **index.html** — Main page with hero, features, how it works, benefits, and contact form
- **styles.css** — Dark theme, professional safety-focused design
- **script.js** — Mobile nav, animated density indicator, form handling
